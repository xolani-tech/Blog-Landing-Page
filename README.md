# MyBlog Landing Page

A responsive blog landing page built with HTML, CSS, and a bit of JavaScript. The design includes a hero section, blog posts, categories, an About section, and a contact form with social icons. The page is mobile-friendly and features smooth scrolling navigation.

- **Hero Section:** Eye-catching welcome section with a call-to-action button.  
- **Blog Section:** Displays blog posts with images, titles, summaries, and "Read More" links.  
- **Categories:** Card-based section for different blog categories.  
- **About Section:** Includes profile image, bio, and personal information.  
- **Contact Form:** Email and message form with contact info and clickable social icons (LinkedIn, GitHub, Portfolio).  
- **Responsive Design:** Works on desktop, tablet, and mobile devices.  
- **Smooth Scroll:** Navbar links scroll smoothly to sections on the same page.  

## Technologies Used

- HTML5  
- CSS3 (Flexbox for layout, responsive design)  
- Google Fonts  
- Font Awesome for icons
